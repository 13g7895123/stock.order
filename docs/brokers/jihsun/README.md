# 日盛證券 (JihSun Securities) API 文件

## 券商資訊

- **券商名稱**: 日盛證券股份有限公司
- **英文名稱**: JihSun Securities Co., Ltd.
- **英文縮寫**: JihSun
- **API 名稱**: 日盛 HTS API

## API 官方資源

- **日盛證券官網**: https://www.jihsun.com.tw/
- **日盛期貨**: 需透過日盛期貨申請

## 主要特色

### 1. 支援的市場
- 台灣期貨市場
- 台灣選擇權市場

### 2. API 類型
- **HTS API**: 主要的 API 介面
- 需搭配 HTS 系統使用

### 3. 核心功能
- 期貨下單
- 選擇權交易
- 自動交易
- 程式交易

## 技術規格

### 系統需求
- Windows 作業系統
- 需安裝 HTS 系統
- .NET Framework 2.0 或以上版本

### 憑證要求
- 需安裝期貨商發給的中繼憑證
- 需安裝數位憑證
- 確保憑證有效性

### API 元件
- HTS API 應用程式介面
- APITradeMgr (API 交易管理程式)

## 申請流程

### 1. 開立日盛期貨帳戶
- 需有日盛期貨帳戶
- 完成開戶流程

### 2. 安裝 HTS 系統
- 下載並安裝最新版本的 HTS
- 確認系統正常運作

### 3. 安裝憑證
- 安裝期貨商發給的中繼憑證
- 安裝個人數位憑證
- 確認憑證有效期限

### 4. 申請 API 權限
- 聯繫日盛期貨營業員
- 申請 API 使用權限
- 簽署相關文件

### 5. 取得 API
- 下載 HTS API 元件
- 安裝 APITradeMgr

### 6. 開始開發
- 參考 API 文件
- 開發自動交易程式

## 每日操作流程

進行 API 下單的每日操作流程：

1. **開啟 HTS 買賣訊號**
   - 確認 HTS 系統正常運作

2. **開啟 APITradeMgr**
   - 啟動 API 交易管理程式

3. **開啟自備下單機**
   - 執行自行開發的下單程式

4. **開始交易**
   - 系統自動執行交易策略

## 使用限制與注意事項

### 1. 資料時效性
- 網路上的資料較舊（約 2010 年）
- 建議直接聯繫日盛證券確認最新資訊

### 2. 平台限制
- 主要支援 Windows
- 需搭配 HTS 系統

### 3. 技術要求
- 需具備程式開發能力
- 需熟悉 .NET Framework
- 需了解期貨交易規則

### 4. 每日操作
- 需每日手動啟動相關程式
- 需確認憑證有效性
- 需監控系統運作狀態

### 5. 費用
- API 使用費用需向營業員確認
- 交易手續費依照帳戶約定費率

## STS 系統

### STS + API 搭配
- STS (日盛交易系統) 可與 API 搭配
- 達到自動下單的完全狀態
- 有紀律、有策略地進行期權交易
