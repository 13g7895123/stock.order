- **券商名稱**: 國泰綜合證券股份有限公司
- **英文名稱**: Cathay Securities Corporation
- **英文縮寫**: Cathay
- **API 名稱**: 國泰 API

## 主要特色

### 1. 支援的市場
- 台灣證券市場
- 台灣期貨市場
- 選擇權市場

### 2. CaaS 服務
- Cathay as a Service (CaaS)
- 國泰金控於 2021 年 7 月推出
- 一站式數位金融服務平台
- 透過 API 提供數位金融服務

## 申請流程

### 1. 開立國泰證券帳戶
- 需有國泰證券交易帳戶

### 2. 聯繫營業員
- 向您的國泰營業員詢問 API 服務
- 了解申請條件和流程

### 3. 簽署文件
- 簽署 API 電子交易風險預告書
- 簽署使用同意書

### 4. 取得 API 文件
- 申請核准後取得 API 元件和文件
- 開始開發測試

## 使用限制與注意事項

### 1. 資訊有限
- 公開的 API 文件較少

### 2. 申請方式
- 建議直接聯繫國泰證券營業員
- 詢問最新的 API 服務內容

### 3. 費用
- API 使用費用需向營業員確認
- 交易手續費依照帳戶約定費率
