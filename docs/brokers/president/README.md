## 申請流程

### 證券 API

1. **開立統一證券帳戶**
   - 需有統一證券交易帳戶

2. **申請 API 服務**
   - 聯繫統一證券營業員
   - 提出 API 使用申請

3. **簽署文件**
   - 簽署相關風險預告書
   - 簽署 API 使用同意書

4. **下載 API**
   - 取得 API 元件
   - 下載範例程式和規格文件

5. **開始開發**
   - 參考 C# 範例程式
   - 依據規格文件開發

### 期貨 API

1. **開立統一期貨帳戶**
   - 需有統一期貨帳戶

2. **申請 PFCF API**
   - 透過統一期貨申請

3. **下載安裝**
   - 下載 PFCF API 元件
   - 參考 GSnail 安裝教學

## 使用限制與注意事項

### 1. 平台限制
- 主要支援 Windows
- 基於 .NET Framework

### 2. 技術要求
- 需具備 C# 或 VB 開發能力
- 熟悉 .NET 開發環境

### 3. API 分離
- 證券 API 和期貨 API 需分別處理
- 期貨使用 PFCF API

### 4. 費用
- API 使用免費
- 交易手續費依照帳戶約定費率

## 常見問題 FAQ

### Q1: 統一證券 API 支援哪些程式語言？
A: 主要支援 C# 和 VB.NET

### Q2: 有 Python 支援嗎？
A: 官方主要提供 .NET 元件，Python 支援有限

### Q3: 證券和期貨 API 一樣嗎？
A: 不一樣，期貨使用 PFCF API，需分別申請

### Q4: 如何取得下單測試紀錄？
A: 可透過統一系統查詢，參考相關教學文章

### Q5: 支援 Mac/Linux 嗎？
A: 官方主要支援 Windows，其他平台可能需要 Wine 或虛擬機
