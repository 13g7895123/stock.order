# 康和證券 (Concord Securities) API 文件

## 券商資訊

- **券商名稱**: 康和綜合證券股份有限公司
- **英文名稱**: Concord Securities Co., Ltd.
- **英文縮寫**: Concord / CSC
- **API 名稱**: 康和期貨 API

## API 官方資源

- **康和證券官網**: https://www.concords.com.tw/
- **康和期貨官網**: https://www.concordfutures.com.tw/
- **程式小幫手**: https://stock.concords.com.tw/trade_071.html
- **API 下載**: https://www.concordfutures.com.tw/ConcordFutures/TradingPlatformDownload/TradeAPI/

## 主要特色

### 1. 支援的市場
- 證券股票
- 國內期貨
- 國外期貨
- 國內選擇權

### 2. API 版本
康和提供 **2 款 API**：

#### 新版 OCX API
- 較新的 API 版本
- 功能較完整

#### 超級飆速 API
- 強調速度
- 適合高頻交易

### 3. 支援的程式語言
- **Excel VBA**
- **C#**
- **VB.NET**

### 4. 核心功能
- 證券下單
- 期貨下單
- 選擇權交易
- 串接第三方程式（如 MultiCharts）
- 自動下單機整合

## 技術規格

### 系統需求
- Windows 作業系統
- .NET Framework
- Excel (如使用 VBA)

### API 元件
- OCX API 元件
- 超級飆速 API 元件
- 範例程式
- 操作說明文件

## 申請流程

### 1. 開立康和期貨帳戶
- 需有康和期貨交易帳戶
- 完成開戶流程

### 2. 電腦線上簽署

透過電腦簽署網址：
- **網址**: https://etradeweb.concords.com.tw/EDeskWeb/ESign/Login.html
- **文件**: 簽署【申請API服務下單聲明書 (期貨客戶適用)】

### 3. 下載 API
- 從康和期貨官網下載
- 選擇適合的 API 版本（OCX 或超級飆速）

### 4. 安裝與設定
- 安裝 API 元件
- 參考操作說明設定

### 5. 開始開發
- 使用 VBA、C# 或 VB.NET
- 參考範例程式開發

## API 版本比較

### 新版 OCX API vs 超級飆速 API

| 特色 | OCX API | 超級飆速 API |
|------|---------|-------------|
| 版本 | 較新 | 強調速度 |
| 適用 | 一般程式交易 | 高頻交易 |
| 功能 | 完整 | 速度優化 |

詳細比較請參考：
- https://gcfc.com.tw/康和期貨程式api/
- https://futures.com.tw/康和期貨api/

## 使用限制與注意事項

### 1. 平台限制
- 主要支援 Windows
- 需要 .NET 環境

### 2. 技術要求
- 需具備 VBA、C# 或 VB.NET 開發能力
- 需了解期貨交易規則
- 需具備程式除錯能力

### 3. 使用資格
- 需有康和期貨帳戶
- 需線上簽署 API 服務聲明書
- 需了解程式交易風險

### 4. 金管會規定
- 2015 年 7 月金管會開放
- 投資人可使用證券商提供的 API 下單

### 5. 費用
- API 使用費用需向營業員確認
- 交易手續費依照帳戶約定費率

## 第三方整合

### MultiCharts
- 康和 API 可串接 MultiCharts
- 用於策略回測和自動交易

### 自動下單機
- 可串接第三方自動下單程式
- 實現完全自動化交易

## 常見問題 FAQ

### Q1: 康和 API 支援哪些商品？
A: 證券股票、國內期貨、國外期貨、國內選擇權

### Q2: 兩款 API 有什麼差別？
A: OCX API 較新功能完整，超級飆速 API 強調速度

### Q3: 支援哪些程式語言？
A: Excel VBA、C#、VB.NET

### Q4: 如何選擇 API 版本？
A: 一般交易選 OCX API，高頻交易可選超級飆速 API

### Q5: 需要什麼帳戶？
A: 需要康和期貨帳戶

### Q6: 如何申請？
A: 透過電腦線上簽署【申請API服務下單聲明書】
